# Estudos de JavaScript

Este repositório foi criado para estudar **JavaScript de forma organizada**, separando os conceitos em arquivos diferentes para facilitar o entendimento e a prática.

## Objetivo
- Praticar lógica de programação
- Reforçar conceitos fundamentais de JavaScript
- Desenvolver o hábito de organizar código em módulos
- Ganhar confiança escrevendo código sozinho
